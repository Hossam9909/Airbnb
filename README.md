# Rental Property Analysis

## Overview

This program analyzes rental property data from two cities: Boston and Seattle. The primary objective is to identify the best places to rent based on various features provided in the datasets. By utilizing linear regression, the program aims to predict rental prices and provide insights into the rental market.

## Libraries Used

The following Python libraries are utilized in this project:

- **NumPy**: For numerical operations.
- **Pandas**: For data manipulation and analysis.
- **Matplotlib**: For data visualization.
- **Seaborn**: For enhanced data visualization.
- **Scikit-learn**: For machine learning tasks, including linear regression, model evaluation, and data preprocessing.

## Data Files

The program requires two CSV files located in the same directory as the script:

1. `Boston.csv`: Contains rental property data for Boston.
2. `Seattle.csv`: Contains rental property data for Seattle.

## How to Run the Program

1. **Install Required Libraries**: Ensure that you have the required libraries installed. You can install them using pip:

   ```bash
   pip install numpy pandas matplotlib seaborn scikit-learn

2. **Place Data Files: Make sure that Boston.csv and Seattle.csv are located in the same directory as the script.

3. **Run the Script: Execute the script in your Python environment. You can use an IDE like PyCharm, Jupyter Notebook, or run it in a terminal.
